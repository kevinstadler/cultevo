Perform Page's test on the given data set, providing exact p values whre possible.

Usage

pages.test(data, ...)

Arguments

data
a matrix or dataframe with the different conditions along columns and the replications along rows. Conversion of the data to ranks is taken care of internally.
...
extra parameters which are passed on to pagesL

Perform Page's test on the given data set, providing exact p values whre possible.

Value

a list of class PagesTest

Description

Page's test tests for monotonically increasing ranks across linearly ordered conditions. To test for monotonically decreasing trends you can either reverse the order of columns, or simply invert the ranks (i.e. by prepending the dataset with a -).

Examples

pages.test(t(replicate(4, sample(4))))



Page's test: L=103, k=4, N=4, p=3.374898e-01 (exact)



pages.test(t(replicate(4, sample(10))))



Page's test: L=1264, k=10, N=4, p=1.678337e-01 (exact)
