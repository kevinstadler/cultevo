Read data from a tab or comma-separated file and run Mantel tests on the data.

Usage

mantel.file(filename = NULL, sep = "\t", header = FALSE, stringcolumns = 1, meaningcolumns = -stringcolumns, ...)

Arguments

filename
csv file to be read - not specifying a file will result in a file selection dialog being opened
header
whether the csv file contains a header line or not
...
extra arguments are passed on to mantel.development (and potentially further to plot.mantels)

Read data from a tab or comma-separated file and run Mantel tests on the data.

Description

Reads the given .csv file and runs Mantel tests on it, optionally plotting the results. Plotting is controlled solely through the ... arguments which are passed on to mantel.development and plot.mantels respectively. If none of the column-arguments are specified it is assumed that the first column gives the strings and all remaining columns encode the different meanings. See mantelexample.csv for a minimal example.

Examples

mantel.file(system.file("minimalexample.csv", package="mantel"), plot="r")


Permutation space is small, enumerating all possible permutations.
