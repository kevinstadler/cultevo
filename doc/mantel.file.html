Read data from a tab or comma-separated file and run Mantel tests on the data.

Usage

mantel.file(filename = NULL, sep = "\t", header = FALSE, stringcolumns = 1, meaningcolumns = -c(stringcolumns, generationcolumn), generationcolumn = NULL, test.args = NULL, ...)

Arguments

filename
csv file to be read - not specifying a file will result in a file selection dialog being opened
header
whether the csv file contains a header line or not
stringcolumns
name(s) or index(es) of the columns containing the strings
meaningcolumns
names or indices of the columns specifying the meanings
generationcolumn
if the strings from consecutive generations are entered along rows rather than columns, this parameter indicates the column which specifies the generation to which a row belongs. When this parameter is specified, stringcolumns should be a single index/name.
test.args
a list of named arguments passed on to mantel.test
...
extra arguments are passed on to mantel.development (and potentially further to mantel.test and plot.mantel)

Read data from a tab or comma-separated file and run Mantel tests on the data.

Value

a dataframe of class mantel specifying the results of the Mantel test(s)

Description

Reads the given .csv file and runs Mantel tests on it, optionally plotting the results. Plotting is controlled solely through the ... arguments which are passed on to mantel.development and plot.mantel respectively. If none of the column-arguments are specified it is assumed that the first column gives the strings and all remaining columns encode the different meanings. See mantelexample.csv for a minimal example.
