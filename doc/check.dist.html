Check or fix a symmetric distance matrix.

Usage

check.dist(m)

Arguments

m
a distance matrix

Check or fix a symmetric distance matrix.

Value

a symmetric matrix of the same size as m, with 0s in the diagonal

Description

If m is a matrix, check whether it is a valid specification of a distance matrix and return it, making it symmetric if it isn't already. For all other obect types, try to coerce m to a dist object and return the corresponding distance matrix.
