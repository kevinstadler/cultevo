Calculate exact significance levels for Page's L.

Usage

pages.test.exact(k, N, p.lvls = c(0.05, 0.01, 0.001), message.progress = k * N >= 30, L = 0)

Arguments

k
number of conditions/generations
N
number of replications/chains
p.lvls
vector of p levels for which the exact critical L value is sought
message.progress
logical, whether to message() information on the progress of the computation to the output
L
the lowest L for which the p value should be computed

Calculate exact significance levels for Page's L.

Description

Calculate critical values of Page's L for the given significance levels.

Details

If only one p value is supplied, the function will return the full table of p values for all Ls computed up to that p value, rather than just the critical L for that particular p.
