Calculate the multinomial coefficient (number of permutations of a multiset)

Usage

mcombn(n, ks)

Arguments

n
ks
vector of positive integers specifying the multiplicities of the individual elements. '1' elements can be omitted.

Calculate the multinomial coefficient (number of permutations of a multiset)

Description

Calculates the number of possible orders in which n elements can be drawn when some of the individual elements are identical. Pure R implementation which is fast enough but might fail for large n/k. In this case you probably want to have a look at multinom in the multicool package.

Examples

# only one order for the same sample drawn five times
mcombn(5,5)



[1] 1



# the full 5! possible orders of drawing five unique samples
mcombn(5,rep(1,5))



[1] 120



# sum(ks) should never exceed n, so this will give funny results:
mcombn(5,c(2,4))



[1] 2.5
