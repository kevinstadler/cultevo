Aggregate rows and columns of a symmetric (distance) matrix.

Usage

conflate.rows(m, groups, discard = FALSE)

Arguments

m
a symmetric matrix, can be a matrix or dist object
groups
a list of group indices/ids of the same length as the dimension of the matrix, indicating which rows/columns should be aggregated over

Aggregate rows and columns of a symmetric (distance) matrix.

Value

a matrix with as many rows and columns as there are unique ids in the groups parameter

Description

Conflates the given matrix by either averaging across or discarding rows which are marked by the same group index/id.
