Compute the normalised Levenshtein distances between strings.

Usage

normalisedlevenshteindists(strings)

Arguments

strings
a vector or list of strings

Compute the normalised Levenshtein distances between strings.

Value

a distance matrix of normalised Levenshtein distances between the strings

Description

Compute the normalised Levenshtein distances between strings.

Examples

normalisedlevenshteindists(c("abd", "absolute", "asdasd", "casd"))



               abd  absolute    asdasd
absolute 0.7500000                    
asdasd   0.6666667 0.7500000          
casd     0.5000000 0.8750000 0.5000000
