Object class for storing the results of one or more Mantel tests

Object class for storing the results of one or more Mantel tests

Description

A data frame storing the results of one or more Mantel tests, one test per row. Multiple mantel objects can easily be combined by calling rbind(test1, test2, ...)

Details

The data frame contains the following columns:

method
Character string specifying the type of correlation coefficient used (typically "pearson")
veridical
The correlation coefficient between the distances in the two distance matrices
sample.size
The number of permutations carried out
msample
A list of correlation coefficients calculated from the sample.size permutations of the input matrices
mean
Average correlation coefficient produced by the permutations
sd
Standard deviation of the correlation coefficients
z
Standard score, (veridical-mean)/sd
ks.level
p-value of the Kolmogorov-Smirnov test of normality of msample. If this value is low, sd and z might not be very meaningful.
p.value
Empirical p-value computed from the Mantel test: let ngreater be the number of correlation coefficients in msample greater than or equal to veridical, then p.value is (ngreater+1)(sample.size+1
p.smoothed
The theoretical p-value that would correspond to the standard z score as calculated above. If the ks.level is less than was specified as an argument to mantel.test, p.smoothed will be set to NA.
is.unique.max
Logical, TRUE iff the veridical correlation coefficient is greater than any of the coefficients calculated for the permutations. If this is true, then p.value == 1/(sample.size+1)
