Plot a Mantel test result.

Usage

"plot"(mantels, plot = if (nrow(mantels) == 1) "msample" else "r", xlab = "generation", ...)

Arguments

mantels
a Mantel test result (a data frame of class mantel)
plot
the measure to be visualised, currently supported: "r", "z" and "msample"
...
additional parameters to be passed on to the plotting function

Plot a Mantel test result.

Description

Plots the result of one or more Mantel permutation tests. The exact visualisation used can be controlled with the plot parameter, which currently supports "r", "z" and "msample".

Details

Red error bars or normal distribution fits signify that a Kolmogorov-Smirnov test of the randomised r's was significantly non-normal at the level specified by ks.level in mantel.test. If the veridical r is plotted in blue it means that it was higher than all other r's generated by the permutation test.

See mantel.development for more interesting example plots.

Examples

plot(mantel.test(hammingdists(allmeaningcombinations(c(2,2,2,2))), suppressWarnings(as.dist(1:16))))





plot(mantel.test(hammingdists(allmeaningcombinations(c(2,2,2,2))), suppressWarnings(as.dist(1:16))), plot="z")
