Plot a Mantel test result.

Usage

"plot"(mantels, plot = "r", xlab = "generation", ...)

Arguments

mantels
a Mantel test result (a data frame of class 'mantel')
plot
the measure to be visualised, currently supported: 'r' and 'z'
...
additional parameters to be passed on to the plotting function

Plot a Mantel test result.

Description

Plots the result of one or more Mantel permutation tests.

Examples

plot(mantel.test(hammingdists(allmeaningcombinations(c(2,2,2,2))), suppressWarnings(as.dist(1:16))))
